Taiwan Warrant OpenAPI Scanner

券商回收權證雷達

自動讀取 TWSE/TPEX OpenAPI，尋找低價、成交量存在、剩餘天數足夠，且具備前日回收參考價證據的台灣權證候選。

前往 GitHub 查看啟動方式

Core

核心流程

自動探測 API

讀取 TWSE/TPEX swagger.json，自動尋找權證行情、成交與基本資料 endpoint。

欄位自動整理

將不同來源的代號、名稱、收盤、成交量、標的、履約價與到期日整理成統一格式。

同代號合併

行情資料與基本資料分散在不同 endpoint 時，系統會依權證代號合併再評分。

SQLite 留存

每日掃描結果落地保存，便於回測、追蹤與後續模型訓練。

Evidence

前日回收價證據層

公式 前日回收參考價 = 今日 Close - Change
輸出 今日高於 / 等於 / 低於前日回收價
用途 驗證候選權證是否真的有價差機會

Score

market_maker_score

  • 價格 <= 0.02 加分
  • 委買 >= 0.01 大幅加分
  • 委賣存在加分
  • 成交量存在加分
  • bid/ask 價差小加分
  • 剩餘天數 >= 10 加分
  • 有前日回收參考價加分

Run

本機啟動 Flask APP

python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py

GitHub Pages 只能展示此靜態網站；真正掃描 TWSE/TPEX 的 Flask 後端需要在本機、VPS、Render、Railway 或其他 Python 服務上執行。
